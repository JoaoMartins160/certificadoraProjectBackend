Apresentação ao JavaScript

Introdução ao JavaScript

JavaScript é uma linguagem de programação de alto nível que é amplamente utilizada para desenvolvimento web. Foi criada originalmente para tornar as páginas web interativas, permitindo manipulação dinâmica do conteúdo, validação de formulários, animações, entre outras funcionalidades. Com o tempo, o JavaScript expandiu seu alcance e agora é utilizado tanto no lado do cliente (navegador) quanto no lado do servidor (Node.js).

Sintaxe

A sintaxe do JavaScript é semelhante à de outras linguagens de programação. Aqui estão alguns elementos básicos:

  • Declaração de variáveis: 'var', 'let', 'const'.
  • Estruturas de controle: 'if', 'else', 'switch', 'for', 'while', 'do-while'.
  • Funções: 'function', '=>'.
  • Comentários: '//' para comentários de linha única, '/* */' para comentários de várias linhas.
  • Expressões e operadores: '+', '-', '*', '/', '==', '===', '!=', '!==', '&&', '||', entre outros.

Variáveis e Tipos de Dados

Em JavaScript, as variáveis são usadas para armazenar valores. Existem três palavras-chave para declarar variáveis: var, let e const.

  • 'var': Declara uma variável globalmente ou localmente em uma função, mas seu escopo é determinado pelo contexto de execução atual.
  • 'let': Declara uma variável local no escopo do bloco em que é definida. Ela é restrita ao bloco, instrução ou expressão onde é definida.
  • 'const': Declara uma constante de escopo de bloco, ou seja, seu valor não pode ser alterado após a atribuição inicial.

Quanto aos tipos de dados, JavaScript é uma linguagem de tipagem dinâmica, o que significa que você não precisa declarar explicitamente o tipo de dado de uma variável antes de atribuir um valor a ela. Os tipos de dados em JavaScript incluem:

  • Primitivos: string, number, boolean, undefined, null, symbol (adicionado no ECMAScript 6).
  • Objetos: objetos, arrays, funções, datas, expressões regulares, entre outros.

Estruturas em Funções

As funções são blocos de construção fundamentais em JavaScript que permitem a reutilização de código. Uma função pode aceitar parâmetros (ou argumentos) e retornar um valor.

Além das funções simples, você pode usar estruturas mais complexas, como:

  • Funções aninhadas: Definir funções dentro de outras funções.
  • Funções de callback: Passar funções como argumentos para outras funções.
  • Funções de retorno de chamada: Retornar funções de outras funções.
  • Funções de flecha (Arrow functions): Uma sintaxe mais curta e concisa para definir funções, introduzida no ECMAScript 6.

DOM (Document Object Mode)

Manipulação de DOM

A manipulação do DOM (Document Object Model) é uma parte essencial do desenvolvimento da web, permitindo que os desenvolvedores interajam dinamicamente com elementos HTML, CSS e JavaScript em uma página da web. Aqui está uma explicação detalhada sobre os principais aspectos da manipulação do DOM: seleção de elementos, modificação de conteúdo e manipulação de eventos.

Seleção de Elementos

  • A seleção de elementos é o processo de identificar e obter referências para elementos específicos no DOM para que possam ser manipulados programaticamente.
  • Existem várias maneiras de selecionar elementos no DOM, incluindo seleção por ID ('getElementById'), por classe ('getElementsByClassName', 'querySelectorAll'), por tag ('getElementsByTagName', 'querySelector'), entre outros métodos.
  • Por exemplo, para selecionar um elemento com o ID "exemplo", você pode usar document.getElementById('exemplo').

Modificação de Conteúdo

  • A modificação de conteúdo refere-se à atualização dinâmica do conteúdo dentro dos elementos HTML.
  • Você pode modificar o texto, atributos ou estrutura HTML de um elemento selecionado.
  • Por exemplo, para alterar o texto de um elemento com ID "exemplo", você pode usar 'document.getElementById('exemplo').textContent = 'Novo texto';''.
  • Para modificar atributos, você pode usar 'elemento.setAttribute('atributo', 'valor')' e para remover um atributo 'elemento.removeAttribute('atributo')'.
  • Para modificar a estrutura HTML, você pode usar métodos como 'innerHTML', 'appendChild', 'removeChild', etc.

Manipulação de Eventos

  • A manipulação de eventos permite que os desenvolvedores respondam a ações do usuário, como cliques, movimentos do mouse, pressionamentos de teclas, etc.
  • Você pode atribuir manipuladores de eventos a elementos específicos no DOM para executar ações quando o evento ocorrer.
  • Os eventos podem ser registrados usando métodos como addEventListener ou atribuindo diretamente os manipuladores de eventos às propriedades dos elementos, como 'onclick'.
  • Por exemplo, para executar uma função quando um botão com ID "botao" for clicado, você pode usar 'document.getElementById('botao').addEventListener('click', minhaFuncao);'.

Banco de Dados e SQL Básico

Conceitos de Banco de Dados

Um banco de dados é um sistema de armazenamento organizado que permite a criação, leitura, atualização e exclusão de dados. Alguns tipos de banco de dados incluem:

  • Bancos de Dados Relacionais: Estruturados em tabelas relacionadas umas às outras por chaves estrangeiras. Exemplos incluem MySQL, PostgreSQL, SQL Server, Oracle.
  • Bancos de Dados NoSQL: Projetados para armazenar e recuperar dados de maneira não relacional. Incluem bancos de dados de documentos, de chave-valor, de coluna larga e de grafo. Exemplos incluem MongoDB, Cassandra, Redis.
  • Bancos de Dados em Memória: Armazenam dados primariamente na memória principal para acesso rápido. Exemplos incluem Redis, Memcached.
  • Bancos de Dados Orientados a Objetos: Armazenam objetos, ao invés de dados em tabelas. Exemplos incluem db4o, ObjectDB.

Além disso, a modelagem de dados é o processo de projetar a estrutura de um banco de dados, incluindo a definição de entidades, atributos e relacionamentos. Aqui estão alguns exemplos de modelagem de dados:

  • Modelo Conceitual: Descreve entidades, seus atributos e relacionamentos, usando conceitos de alto nível. Exemplo: Modelo Entidade-Relacionamento (ER).
  • Modelo Lógico: Descreve a estrutura de dados sem considerar implementações específicas. Exemplo: Modelo Relacional.
  • Modelo Físico: Descreve a estrutura de dados considerando implementações físicas específicas. Exemplo: Esquema de Banco de Dados.

SQL Básico

SQL (Structured Query Language) é uma linguagem de consulta estruturada utilizada para interagir com bancos de dados relacionais. Algumas operações básicas em SQL incluem:

Consultas SELECT:

  • Utilizadas para recuperar dados de um banco de dados.
  • Sintaxe básica: 'SELECT coluna1, coluna2 FROM tabela WHERE condição;'.
  • Exemplo: 'SELECT nome, idade FROM usuarios WHERE cidade = 'Nova York';'.

Inserção de Dados:

  • Utilizada para adicionar novos registros a uma tabela.
  • Sintaxe básica: 'INSERT INTO tabela (coluna1, coluna2) VALUES (valor1, valor2);'.
  • Exemplo: 'INSERT INTO usuarios (nome, idade) VALUES ('João', 30);'.

Atualização de Dados:

  • Utilizada para modificar registros existentes em uma tabela.
  • Sintaxe básica: 'UPDATE tabela SET coluna1 = valor1, coluna2 = valor2 WHERE condição;'.
  • Exemplo: 'UPDATE usuarios SET idade = 31 WHERE nome = 'João';'.

Exclusão de Dados:

  • Utilizada para remover registros de uma tabela.
  • Sintaxe básica: 'DELETE FROM tabela WHERE condição;'.
  • Exemplo: 'DELETE FROM usuarios WHERE nome = 'João';'.
