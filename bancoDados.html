Banco de Dados e SQL Básico

Conceitos de Banco de Dados

Um banco de dados é um sistema de armazenamento organizado que permite a criação, leitura, atualização e exclusão de dados. Alguns tipos de banco de dados incluem:

  • Bancos de Dados Relacionais: Estruturados em tabelas relacionadas umas às outras por chaves estrangeiras. Exemplos incluem MySQL, PostgreSQL, SQL Server, Oracle.
  • Bancos de Dados NoSQL: Projetados para armazenar e recuperar dados de maneira não relacional. Incluem bancos de dados de documentos, de chave-valor, de coluna larga e de grafo. Exemplos incluem MongoDB, Cassandra, Redis.
  • Bancos de Dados em Memória: Armazenam dados primariamente na memória principal para acesso rápido. Exemplos incluem Redis, Memcached.
  • Bancos de Dados Orientados a Objetos: Armazenam objetos, ao invés de dados em tabelas. Exemplos incluem db4o, ObjectDB.

Além disso, a modelagem de dados é o processo de projetar a estrutura de um banco de dados, incluindo a definição de entidades, atributos e relacionamentos. Aqui estão alguns exemplos de modelagem de dados:

  • Modelo Conceitual: Descreve entidades, seus atributos e relacionamentos, usando conceitos de alto nível. Exemplo: Modelo Entidade-Relacionamento (ER).
  • Modelo Lógico: Descreve a estrutura de dados sem considerar implementações específicas. Exemplo: Modelo Relacional.
  • Modelo Físico: Descreve a estrutura de dados considerando implementações físicas específicas. Exemplo: Esquema de Banco de Dados.

SQL Básico

SQL (Structured Query Language) é uma linguagem de consulta estruturada utilizada para interagir com bancos de dados relacionais. Algumas operações básicas em SQL incluem:

Consultas SELECT:

  • Utilizadas para recuperar dados de um banco de dados.
  • Sintaxe básica: 'SELECT coluna1, coluna2 FROM tabela WHERE condição;'.
  • Exemplo: 'SELECT nome, idade FROM usuarios WHERE cidade = 'Nova York';'.

Inserção de Dados:

  • Utilizada para adicionar novos registros a uma tabela.
  • Sintaxe básica: 'INSERT INTO tabela (coluna1, coluna2) VALUES (valor1, valor2);'.
  • Exemplo: 'INSERT INTO usuarios (nome, idade) VALUES ('João', 30);'.

Atualização de Dados:

  • Utilizada para modificar registros existentes em uma tabela.
  • Sintaxe básica: 'UPDATE tabela SET coluna1 = valor1, coluna2 = valor2 WHERE condição;'.
  • Exemplo: 'UPDATE usuarios SET idade = 31 WHERE nome = 'João';'.

Exclusão de Dados:

  • Utilizada para remover registros de uma tabela.
  • Sintaxe básica: 'DELETE FROM tabela WHERE condição;'.
  • Exemplo: 'DELETE FROM usuarios WHERE nome = 'João';'.
